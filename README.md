# 人民币大写转换器（RMB Converter）

一个轻量级 Chrome 浏览器扩展，将小写数字金额快速转换为中文大写人民币写法，支持剪贴板一键复制。

## 功能特性

- 输入小写金额自动转换
- 支持小数（精确到分）
- 一键复制结果
- 离线可用，无需网络

## 安装方法

1. 下载代码或 zip
2. 打开 Chrome 地址栏输入 `chrome://extensions/`
3. 开启右上角「开发者模式」
4. 点击「加载已解压的扩展程序」选择本目录

## 使用说明

点击工具栏图标，输入金额数字，自动显示大写结果，点击复制按钮即可。

## 技术栈

- Manifest V3
- HTML / CSS / JavaScript
